
# Lincos Translation Dataset

## Описание
Датасет для перевода с русского языка на Lincos - универсальный язык для космической коммуникации.

## Статистика
- Всего примеров: 3229
- Тренировочных: 2583
- Тестовых: 646
- Категории: {'logic': np.int64(1500), 'math': np.int64(1231), 'comparison': np.int64(498)}

## Структура
Каждый пример содержит:
- russian: текст на русском
- lincos: перевод на Lincos
- category: категория (math/logic/comparison)
- difficulty: уровень сложности (1-3)
